# win-path-convert

一个轻量级的 Windows 剪贴板路径转换工具，可以自动将 Windows 格式的路径（使用反斜杠 `\`）转换为 Unix 格式（使用正斜杠 `/`）。

## 功能特点

- 🔄 **自动转换**：监听剪贴板变化，自动转换复制的路径格式
- 🎯 **智能识别**：能够智能识别需要转换的内容，避免误转换 URL 等内容
- 🔧 **双模式运行**：支持剪贴板监听模式和轮询模式，适应不同系统环境
- 📋 **排除规则**：支持自定义排除模式，防止特定格式内容被错误转换
- 📝 **详细日志**：提供多级别日志输出，便于调试和监控
- 🚀 **轻量高效**：资源占用低，不影响系统性能

## 工作原理

程序在后台运行，监听剪贴板内容变化。当检测到包含 Windows 路径格式的内容被复制到剪贴板时，会自动将其转换为 Unix 格式，然后放回剪贴板。这样当你在其他应用程序中粘贴时，会得到兼容的 Unix 格式路径。

下载文件中的示例展示了转换效果，包括各种类型的路径和环境变量。

## 系统要求

- **操作系统**：Windows 10/11 (推荐)
- **架构**：支持 x64 和 x86 架构
- **依赖**：无需额外依赖，可直接运行

## 安装方法

### 方法一：下载预编译版本（推荐）

1. 访问项目的 [Releases 页面](https://github.com/lyj404/win-path-convert/releases)
2. 下载最新版本的 `win-path-convert.zip`
3. 解压缩到任意文件夹
4. 在命令行中运行 `win-path-convert.exe`

### 方法二：使用 Go 编译（开发者）

1. 安装 [Go](https://golang.org/dl/) 版本 1.25.1 或更高
2. 克隆仓库：
   ```bash
   git clone https://github.com/lyj404/win-path-convert.git
   cd win-path-convert
   ```
3. 编译程序：
   ```bash
   go build -o win-path-convert.exe ./cmd/main.go
   ```
4. 复制 `win-path-convert.exe` 到您希望的位置，并在命令行运行

## 使用方法

### 基本使用

1. 在命令行中运行 `win-path-convert.exe`
2. 复制任何包含反斜杠的 Windows 路径
3. 程序会自动将其转换为正斜杠格式
4. 在需要的地方粘贴，得到转换后的路径

## 常见问题

### 如何退出程序

- 在命令行中使用 `Ctrl+C` 终止程序
